RFP Response Pipeline

A sequential 13-command workflow that produces all required deliverables for a professional RFP response, from intelligence gathering to animated architecture diagrams.

Command Purpose & Deliverables Output Format
/rfp-analyze Deep-read the RFP, extract intelligence, and perform a Go/No-Go assessment. .md .docx
/rfp-requirements Extract and organize every requirement into a structured, traceable format. .md .docx
/rfp-instructions Generate a section-by-section writing guide for the proposal team. .md
/rfp-estimate Calculate effort estimate, role-by-role hours, and staffing plan. .md
/rfp-detailed-estimate Granular, requirement-level effort decomposition with full traceability. .xlsx
/rfp-pricing Generate commercial pricing with rate card and margin analysis. .xlsx
/rfp-respond Write the full 16-section proposal document following Aiden AI standards. .md .docx
/rfp-deck Generate a professional summary presentation (executive deck). .html .pptx
/rfp-clarifications Generate structured clarification questions for the client. .xlsx
/rfp-architecture Create technical architecture SVGs and a single-page HTML viewer. .svg .html
/rfp-support-estimate Estimate and price application support models (24x7/16x5/8x5). .xlsx
/rfp-prototype Generate interactive UI prototypes (Streamlit/Gradio/HTML). .py .html
/rfp-animated-diagram Create animated architecture flow GIFs with moving data paths. .gif .py
/rfp-humanize Rewrite proposal narrative to pass AI detectors and sound natural. .md .docx
/rfp-full Shortcut: Runs all commands in the pipeline end-to-end. ALL
